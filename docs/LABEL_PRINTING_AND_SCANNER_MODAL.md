# Cabeçalho, scanner e impressão de etiquetas

## Cabeçalho

No painel administrativo, a marca da organização aparece somente como
logomarca.

O nome, a razão social e o CNPJ continuam disponíveis nos locais em que são
necessários, como configurações e documentos.

## Resultado do scanner

Quando uma etiqueta é identificada, o Ativelo abre um modal com:

- status;
- número patrimonial;
- nome;
- número de série;
- Service Tag;
- hostname;
- responsável.

O usuário pode:

- abrir a ficha completa do ativo;
- fechar o resultado;
- iniciar a leitura de outra etiqueta.

## Impressão individual

A etiqueta individual permite escolher:

- tamanho;
- uma a vinte cópias.

As cópias são organizadas automaticamente em folhas A4.

## Impressão em lote

A tela de ativos possui o botão:

```text
Imprimir etiquetas
```

Filtros disponíveis:

- busca;
- categoria;
- status;
- unidade;
- prédio;
- andar;
- setor;
- sala.

O usuário pode:

- selecionar todos os ativos filtrados;
- desmarcar todos os ativos filtrados;
- selecionar equipamentos individualmente;
- definir cópias para todos os selecionados;
- ajustar cópias de cada equipamento;
- escolher tamanho compacto, padrão ou grande;
- visualizar quantidade total de etiquetas;
- visualizar estimativa de folhas A4.

## QR Code

O conteúdo permanece compacto:

```text
ATV1:NUMERO-PATRIMONIAL
```

As etiquetas usam:

- preto puro;
- fundo branco;
- margem de quatro módulos;
- correção de erro alta;
- código patrimonial visível.
